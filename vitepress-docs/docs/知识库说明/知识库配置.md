# 知识库配置

创建完成后，可在私域知识库列表页对知识库进行管理与配置。

![知识库列表](/assets/私域知识库/私域知识库-配置.png)

列表中每个知识库卡片显示以下信息：

| 字段 | 说明 |
|------|------|
| 知识库名称 | 知识库的显示名称及解析模式标签 |
| 描述 | 知识库的用途说明 |
| 文件数 | 已上传的文档数量 |
| 字符数 | 知识库中已索引的总字符数 |
| 关联应用数 | 当前引用该知识库的 AI 应用数量 |

---

## 编辑配置

点击任意知识库卡片，进入该知识库的**编辑配置页面**。

![知识库编辑配置](/assets/私域知识库/私域知识库-编辑配置.png)

页面提供以下操作：

| 操作 | 说明 |
|------|------|
| 文件上传 | 向知识库追加上传新文档 |
| 召回测试 | 输入测试内容，验证知识库的检索效果，详见[召回测试](./召回测试) |
| 设置 | 调整知识库的检索模式与可见权限（见下方） |
| 授权设置 | 配置知识库中文件的访问授权范围，详见[文件授权](./文件授权) |

文件列表展示已入库的所有文档，包含文件名、字符数、上传时间、状态、授权对象等信息，支持对单个文件进行启用/禁用或删除操作。

---

## 设置

在编辑配置页面点击**设置**按钮，可重新调整知识库的核心配置。

![知识库设置](/assets/私域知识库/私域知识库-设置.png)

### 可见权限

控制该知识库对团队成员的可见范围：

| 选项 | 说明 |
|------|------|
| 只有我 | 仅创建者本人可见 |
| 所有团队成员 | 团队内所有成员均可查看和使用 |
| 自定义 | 手动指定可访问的成员或部门 |

### 索引方式

| 选项 | 说明 |
|------|------|
| 高质量 | 调用系统默认嵌入接口处理，查询准确度更高，会消耗 Token |
| 经济 | 使用离线向量引擎、关键词索引等方式，降低准确度但无需花费 Token |

### 索引增强

可在此处开启或关闭索引增强功能，详细说明请参考[创建知识库 - 索引增强](./创建知识库#索引增强)。

### Embedding 模型

选择用于将文本转换为向量的嵌入模型，如 `text-embedding-3-large`，影响向量检索的语义理解能力。

### 检索设置

调整知识库的检索模式与参数：

| 检索模式 | 说明 |
|----------|------|
| 向量检索 | 通过生成查询嵌入并查找与其语义最相似的文本分段 |
| 全文检索 | 索引文档中的所有词汇，允许用户查询任意词汇并返回包含这些词汇的文本片段 |
| 混合检索 | 同时执行全文检索和向量检索，并应用重排序步骤，需配置 Rerank 模型 API |

向量检索模式下可配置以下参数：

| 参数 | 说明 |
|------|------|
| Rerank 模型 | 对检索结果进行二次排序，提升最终返回结果的相关性 |
| Top K | 返回的最大切片数量，数值越大召回内容越多 |
| Score 阈值 | 相似度得分的最低门槛，低于该值的切片将被过滤，不参与返回 |

配置完成后点击右上角**保存**生效。
