# 文件授权

文件授权用于控制知识库中文件的访问权限，支持将文件授权给指定用户或角色，并精细设置其操作权限（仅引用或引用及下载），实现文件级别的权限管理。

---

## 进入文件授权

在知识库编辑配置页面，勾选一个或多个需要授权的文件，点击右上角**授权设置**按钮，在下拉菜单中选择**文件授权**，弹出授权配置弹窗。

![文件授权入口](/assets/私域知识库/文件授权.png)

> **提示：** 支持批量勾选多个文件后统一授权，适合需要对同一批文件设置相同权限的场景。

---

## 授权配置

![文件授权配置](/assets/私域知识库/文件授权-权限分配.png)

### 用户类型

弹窗顶部切换授权对象的类型：

| 类型 | 说明 |
|------|------|
| 按用户 | 将文件授权给指定的个人成员，适合针对特定人员的精细化授权 |
| 按角色 | 将文件授权给某个角色下的所有成员，适合按岗位或职能批量授权 |

### 权限配置

在**权限配置**下拉框中选择本次授权的默认权限类型，后续也可对每个授权对象单独调整：

| 权限类型 | 说明 |
|----------|------|
| 仅引用 | 被授权成员可通过 AI 应用检索并引用该文件的内容，但无法下载原始文件 |
| 引用及下载 | 被授权成员既可检索引用，也可直接下载该文件的原始内容 |

### 选择授权对象

左侧列表展示当前系统中的所有成员（按用户模式）或所有角色（按角色模式），支持关键字搜索快速定位。

- 勾选对应成员或角色后，右侧**授权用户** / **授权角色**区域会实时显示已选中的授权对象
- 支持**全选**一键选中所有成员
- 右侧每条授权记录可单独调整权限类型，点击权限下拉框选择**仅引用**、**引用及下载**或**移除**

### 保存

配置完成后点击**保存**，授权设置立即生效。文件列表中的**授权对象**列会同步显示已授权的用户或组织信息。

---

## 说明

- 文件授权是在知识库可见权限基础上的进一步限制，仅对知识库有访问权限的成员才能被授权
- 未被授权的成员即使所在知识库对其可见，也无法通过 AI 应用检索到该文件的内容
- 同一成员若同时被按用户和按角色授权，以权限较高的配置为准
- 如需批量调整整个知识库的访问范围，建议通过**设置**中的可见权限统一管理，详见[知识库配置 - 可见权限](./知识库配置#可见权限)
