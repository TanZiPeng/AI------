# 词库授权

词库授权用于控制词库的访问权限，分为**维护**和**引用**两种类型，可分别针对常规词库和脱敏词库进行配置。

点击自定义词库页面右上角的**词库授权**按钮，弹出授权配置弹窗。

---

## 授权类型

弹窗顶部通过**类型**标签切换授权模式：

| 类型 | 说明 |
|------|------|
| 维护 | 控制哪些用户可以对词库进行编辑、添加、删除词条等操作 |
| 引用 | 控制哪些用户可以在翻译时引用该词库 |

---

## 维护授权

切换至**维护**标签后，弹窗分为三栏：

![词库授权-维护](/assets/词库说明/词库授权-维护.png)

| 区域 | 说明 |
|------|------|
| 左侧词库列表 | 选择要配置授权的词库，支持**常规**和**脱敏**两个标签页切换 |
| 中间用户/部门选择 | 按部门树形结构展示所有用户，勾选即授权；支持按名称搜索 |
| 右侧已授权用户 | 展示当前已授权的用户列表（手机号 + 姓名），支持单独**删除**，或点击**清空**移除全部授权 |

配置完成后点击**保存**生效，点击**取消**关闭弹窗。

---

## 引用授权

切换至**引用**标签后，配置逻辑与维护授权一致，但控制的是词库的引用权限。

![词库授权引用](/assets/词库说明/词库授权引用.png)

未添加任何授权用户时，中间选择区域显示**暂无数据**，右侧已授权用户数量显示为 **0 个**。

---

## 操作说明

| 操作 | 说明 |
|------|------|
| 勾选用户 | 在中间列表中勾选用户或部门，右侧自动同步显示已授权用户 |
| 删除单个授权 | 点击右侧列表中某用户右侧的**删除**按钮，移除该用户的权限 |
| 清空全部授权 | 点击右侧列表顶部的**清空**按钮，一键移除所有已授权用户 |
| 保存 | 点击**保存**使当前授权配置生效 |

> **提示：** 维护权限和引用权限相互独立，可分别配置不同的授权范围。例如，仅允许特定人员维护词库内容，但允许更多人员在翻译时引用该词库。
