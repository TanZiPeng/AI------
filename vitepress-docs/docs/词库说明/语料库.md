# 语料库

语料库用于管理翻译助手中的专业术语词条，告诉大模型在翻译时遇到特定词汇应输出固定译文。例如，遇到"药理学"时固定翻译为"pharmacology"。对于专业性较强的领域，模型的默认翻译结果可能不符合预期，通过语料库可确保术语翻译的准确性和一致性。

---

## 创建词库

进入**词库说明 → 语料库**页面，点击**创建词库**按钮，填写词库名称后完成创建。

![创建语料库](/assets/词库说明/创建语料库.png)

---

## 添加词条

词库创建完成后，点击**增加词组**按钮，在弹窗中填写源文描述（原文）和译文描述（译文），点击确认保存。

![增加词组](/assets/词库说明/增加词组.png)

| 字段 | 说明 |
|------|------|
| 源文描述 | 原文词汇，如 `药理学`、`毒物学` |
| 译文描述 | 对应的固定译文，如 `pharmacology`、`toxicology` |

---

## 导入词条

支持批量导入词条，点击**导入**按钮，按模板格式上传文件即可批量写入词条，适合词条数量较多的场景。

![导入词库](/assets/词库说明/导入词库.png)
