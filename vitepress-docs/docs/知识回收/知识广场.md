# 知识广场

拥有对应权限后，可在顶部导航栏中看到「知识广场」菜单。



## 库列表

知识广场中包含多个知识库，每个库对应一个 Agent 的知识回收库。
![知识广场入口](/assets/知识回收/知识回收-8.png)点击进入后，可查看每次同步的历史记录。


![知识库同步记录](/assets/知识回收/知识回收-9.png)

---

## 召回测试

在库详情页中可进行**召回测试**，输入不同问题，查看会召回哪些知识片段，以此作为调试 embedding / rerank 阈值的依据。

![召回测试](/assets/知识回收/知识回收-10.png)

---

## 记录管理

返回同步记录列表后，支持以下操作：

- **删除同步记录**：对某次整体同步记录进行删除
- **查看 QA 详情**：点击某条同步记录，可查看该次回收的所有 QA 消息对
- **删除单条 QA**：在详情页中可针对单个 QA 对进行删除操作

![QA 详情管理](/assets/知识回收/知识回收-11.png)
