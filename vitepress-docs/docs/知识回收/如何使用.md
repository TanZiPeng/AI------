# 如何使用回收的知识

## 助手类型应用

知识回收的产物相当于一个**私域知识库**，需要前往「我的助手」应用参数设置页面，将知识库绑定到对应助手。

![知识回收-绑定知识库](/assets/知识回收/知识回收-6.png)

绑定后，当用户提问与回收消息中的问题经过 rerank、embedding 匹配一致时，系统会直接回复对应的 QA 对。

---

## 对话型业务流

每个对话型业务流 Agent **只能绑定自身的知识回收库**，无法绑定其他 Agent 的库。

使用步骤：

1. 打开对应业务流的编排页面
2. 在功能栏中开启「标注回复」功能

![知识回收-标注回复](/assets/知识回收/知识回收-7.png)

---

## 参数说明

| 参数 | 说明 |
|------|------|
| Embedding 模型 | 根据使用场景绑定对应的向量化模型 |
| 分数阈值 | 控制匹配精度，阈值为 `1` 表示完全匹配，值越低匹配范围越宽松 |
