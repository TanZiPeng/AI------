# 触发知识回收

本章节介绍如何通过对话交互触发知识回收流程，将高质量问答沉淀为 QA 知识库。

## 操作步骤

### 第一步：开启知识回收功能

进入**我的助手**，在对话型业务流中搭建智能体，然后在配置界面中启用**知识回收**功能。

![开启知识回收功能](/assets/知识回收/知识回收-1.png)

### 第二步：进行问答交互

与智能体进行正常的问答对话。

![问答交互示例](/assets/知识回收/知识回收-2.png)

### 第三步：点赞满意的回答

当遇到质量较高、值得保留的问答时，对该条消息进行**点赞**操作，系统将自动标记该内容待收录。

> **说明：** 系统会在当日凌晨执行定时任务，统一收集所有已点赞的消息。次日即可前往**知识审核**界面查看并审核回收的内容。
