# 我的助手

我的助手调试界面用于在发布前配置提示词、变量、知识库，并通过右侧调试预览面板实时验证效果。

![调试界面](/assets/调试发布/我的助手-调试界面.png)

---

## 页面布局

页面分为左侧**参数配置区**和右侧**调试预览区**两部分。

---

## 参数配置区

### 提示词

提示词定义了助手的角色定位与行为规则，支持三种来源：

| 来源 | 说明 |
|------|------|
| 全局默认 | 使用系统预设的全局提示词模板 |
| 模型默认 | 使用当前所选模型的默认提示词 |
| 自定义 | 手动编写提示词，完全自定义助手的行为逻辑 |

提示词编辑区支持两种阅读模式切换：

| 模式 | 说明 |
|------|------|
| 基础模式 | 默认模式，直接编辑提示词文本 |
| 全文阅读模式 | 以更易阅读的排版展示提示词内容 |

### 变量表单

点击**添加变量**可在提示词中插入动态变量。每次启动新会话时，系统会自动将变量替换为用户在调试预览区**用户输入**面板中填写的值。

### 知识库

为助手关联私域知识库，使其在回复时可检索知识库内容作为参考依据。

| 操作 | 说明 |
|------|------|
| 添加知识库 | 从已创建的知识库中选择并关联 |
| 参数设置 | 配置知识库的检索策略，如召回数量、相似度阈值等 |

已关联的知识库以卡片形式展示，显示知识库名称、字符数和文档数，支持单独配置或删除。

---

## 调试预览区

右侧调试预览面板用于实时测试助手效果，无需发布即可验证配置是否符合预期。

| 操作 | 说明 |
|------|------|
| 用户输入 | 若配置了变量，在此填写变量值，每次新会话自动替换提示词中的变量 |
| 对话输入框 | 在底部输入框中输入问题，发送后查看助手回复 |
| 重新开始 | 清空当前对话，重新开始一轮新的调试会话 |

---

## 模型选择与发布

页面右上角提供模型下拉选择框，可切换当前调试所用的模型（如 `Claude Sonnet 4.5`）。

配置完成并调试验证通过后，点击右上角**发布**按钮将助手发布上线。

> **提示：** 发布前建议通过调试预览充分测试各类场景，确认提示词和知识库配置符合预期后再发布。
