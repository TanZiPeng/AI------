# 知识库

知识库是基于 **RAG（检索增强生成）** 技术的知识管理模块，支持多格式文档上传、向量化处理、语义检索与智能问答，为智能体助手提供精准的知识支撑，有效降低模型幻觉（Hallucination）。

---

## 创建知识库

### 第一步：填写基本信息并选择数据源

点击「**创建私域知识库**」按钮，进入创建页面，依次完成以下配置：

| 配置项 | 说明 |
|--------|------|
| 知识库名称 | 填写便于识别的名称 |
| 内容描述 | 简要描述知识库的用途和内容范围 |
| 切片模式 | 选择「标准模式」或 「高级解析模式模式」|
| 上传文档 | TXT、HTML、Markdown、PDF、XLSX档等多格式 |
| 可见权限 | 设置知识库的访问范围（只有我/所有团队成员/自定义） |

![知识库1](/assets/知识库/知识库1.png)

::: tip 切片模式说明
- **标准模式**：适用于结构清晰的纯文本文档，处理速度快。
- **高级解析模式**：适用于含图表、表格的复杂文档，需在「模型管理 → 模型配置」中提前配置**文档解析服务**。
:::

---

### 第二步：文本分段与清洗

上传文档后，系统将对文本进行分段处理。您可根据实际需求选择以下配置：

**分段策略**

- **自动分段与清洗**：系统自动识别段落边界，适合大多数场景。
- **自定义分段**：手动设置分隔符、分段长度及重叠字符数，适合对分段精度有较高要求的场景。

**图像处理（仅高级解析模式）**

选择图像处理模型，并填写对应的提示词，用于提取文档中图表、截图等非文本内容的语义信息。

![知识库2](/assets/知识库/知识库2.png)

**索引增强**

通过定义和识别索引标签，提升相似度较高内容的检索准确度。适用于零部件批次、产品型号、合同编号等需要精确匹配的场景。

![知识库3](/assets/知识库/知识库3.png)

**检索模式**

| 检索模式 | 适用场景 |
|----------|----------|
| 向量检索 | 语义相似度匹配，适合自然语言问答 |
| 全文检索 | 关键词精确匹配，适合专有名词、编号查询 |
| 混合检索 | 结合向量与全文检索，兼顾语义与精确度（推荐） |

---

### 第三步：文档处理与状态确认

文档上传后，系统将自动进行向量化处理。处理完成后，文档状态变更为「**可用**」。

![知识库4](/assets/知识库/知识库4.png)


若文档状态长时间停留在「索引中」，请检查文档解析服务配置是否正确，或尝试重新上传。


---

## 召回测试

所有文档状态变为「可用」后，点击左上角「**召回测试**」按钮，进入测试页面。

输入测试问题，系统将返回匹配的文档分片列表，并按相关度排序，帮助您验证知识库的检索效果。

![知识库5](/vitepress-docs/docs/assets/知识库/知识库5.png)

**建议在以下情况下进行召回测试：**
- 知识库首次创建完成后
- 新增或更新文档后
- 调整检索模式或分段策略后

---

## 授权管理

知识库支持细粒度的权限控制，管理员可对文档或文件夹按**用户**或**角色**进行授权。

| 权限类型 | 说明 |
|----------|------|
| 引用授权 | 被授权用户可在应用中引用该知识库内容 |
| 引用与下载授权 | 被授权用户可引用内容，并下载原始文档 |

> 只有获得授权的用户或角色，才能在对应应用中引用该知识库，并执行相应的下载操作。
