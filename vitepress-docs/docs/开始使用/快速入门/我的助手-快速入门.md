# 我的助手

**我的助手**是一种对话式应用，采用一问一答模式与用户持续交互，适合构建智能客服、知识问答、业务助理等场景。

## 适用场景

| 场景 | 说明 |
|------|------|
| 客户服务 | 自动响应用户咨询，降低人工客服压力 |
| 在线教育 | 提供课程答疑、学习辅导等交互式服务 |
| 医疗保健 | 辅助健康咨询、症状初筛等场景 |
| 金融服务 | 解答产品政策、引导业务办理流程 |

## 创建应用

进入 **AIGC 应用/应用中心**，点击「创建应用」按钮，填写应用名称，应用类型选择**我的助手**，完成创建。

## 编排参数配置

创建完成后，系统自动跳转至参数配置页，按以下模块依次完成配置。

### 提示词

提示词支持三种来源，优先级依次为：

| 类型 | 说明 |
|------|------|
| 全局默认 | 使用**模型管理 > 模型配置 > 提示词模板**中配置的全局默认提示词，可在应用内覆盖修改 |
| 模型默认 | 使用对应供应商及具体模型的默认提示词，可在应用内覆盖修改 |
| 自定义 | 在提示词模板中预先保存的自定义模板，可通过下拉选择直接复用 |

![我的助手2](/assets/我的助手/助手2.png)

### 知识库

为应用关联知识库，可将对话范围限定在指定知识内容中（如企业文档规范）。

- **添加知识库**：点击「添加」按钮，在弹窗中选择有权限访问的私域知识库或系统知识库，支持多选。
- **参数设置**：点击「参数设置」按钮，配置召回策略。
  - 单知识库：推荐使用 **N 选 1 召回**
  - 多知识库：推荐使用**多路召回**

![我的助手3](/assets/我的助手/助手3.png)

### 组件

点击「关联组件」按钮，在右侧侧边栏中选择已授权的组件并开启，即可在对话中调用该组件能力。

![我的助手4](/assets/我的助手/助手4.png)

### 视觉

> 仅当应用所选基础模型支持视觉能力时，该开关才会显示。

开启后，输入框左侧将出现图片上传入口，用户可上传图片由模型解析图片内容。

### 会话增强

| 功能 | 说明 |
|------|------|
| 私域知识库引用默认开关 | 开启后，助手应用默认启用私域知识库引用 |
| 个人知识库引用默认开关 | 开启后，助手应用默认启用个人知识库引用 |
| 多轮对话默认开关 | 开启后，助手应用默认启用多轮对话上下文 |
| 对话开场白 | 开启后，可配置若干开场引导问题，引导用户快速提问 |
| 下一步问题建议 | 开启后，根据当前问题自动推荐相关追问 |
| 文字转语音 | 开启后，可配置语言、音色（4 种）、语速（4 种）；开启流式输出后，模型边输出文字边播放语音 |
| 语音转文字 | 开启后，申请麦克风权限，支持普通话、粤语、闽南语等语音录入转文字提问 |
| 引用和归属 | 开启且关联知识库后，回答结果下方将展示引用的知识库文档来源 |
| 过滤元数据标签 | 需配合知识库**索引增强**使用，过滤相似度过高的召回片段，提升回复准确性 |

![我的助手5](/assets/我的助手/助手5.png)

### 个人知识库分段 & 检索设置

**分段设置**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 分段最大长度 | 5000 | 单个分段的最大字符数 |
| 分段标识符 | **** | 自定义分段切割标识 |
| 分段重叠长度 | 50 | 相邻分段的重叠字符数，有助于保留上下文 |

> 修改个人知识库分段设置不会影响私域知识库的分段配置。

**检索设置**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 检索模式 | 混合检索 | 结合语义与关键词检索 |
| Rerank 模型 | 模型管理中配置的 Rerank 模型 | 对召回结果重排序 |
| Top K | 3 | 返回相关度最高的前 K 个片段 |
| Score 阈值 | 关闭 | 开启后过滤低于阈值的召回结果 |

> 检索设置仅影响当前应用的个人知识库文档检索，不影响其他应用。
