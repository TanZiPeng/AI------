# 组件中心

组件中心用于管理平台中各类第三方服务组件的授权配置。通过购买或自行部署对应服务后，在此填写服务地址与密钥，即可在工作流、知识库等功能中调用相应能力。

![组件中心首页](/assets/组件中心/组件中心-首页.png)

---

## 授权组件

点击组件卡片上的**授权**按钮，在弹窗中填写以下信息完成授权：

| 配置项 | 是否必填 | 说明 |
|--------|----------|------|
| 服务地址（URL） | 必填 | 组件服务的访问地址，购买或部署后由服务商提供 |
| API Key | 必填 | 用于身份验证的密钥，购买或部署后由服务商提供 |

填写完成后点击**确认**，系统会验证连接是否正常。授权成功后，组件状态变为**已授权**，即可在平台中使用该组件。

![授权组件](/assets/组件中心/组件中心-授权组件.png)

> **提示：** 如需更换服务地址或密钥，重新点击授权按钮覆盖填写即可。

---

## 获取服务地址与 Key

根据使用方式不同，获取方式如下：

- **购买服务**：向服务商购买后，服务商会提供对应的服务地址和 API Key
- **自行部署**：按照服务的部署文档完成部署后，在服务管理后台获取访问地址和密钥

> **注意：** 请妥善保管 API Key，避免泄露。如 Key 失效或更换，需重新在组件中心进行授权。
