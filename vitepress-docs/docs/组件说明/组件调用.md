# 组件调用

组件创建或授权完成后，即可在各类 Agent 智能体应用中调用，包括**我的助手**、**对话型工作流**和**工作流**。

---

## 在我的助手中调用

进入**我的助手**编辑页，在左侧参数配置区找到**组件**模块，点击**关联组件**，右侧弹出组件选择面板。

面板分为三个 Tab：
- **组件**：平台内置的公共组件（如 Google、Bing、Perplexity 等搜索类组件）
- **我的组件**：在组件中心自行创建的自定义组件
- **业务流**：已发布的工作流，可作为组件调用

选择需要的组件后，点击右侧**添加**按钮完成关联。已授权的组件会显示**已授权**标识，未授权的组件需先前往组件中心完成授权。

![我的助手-关联组件](/assets/组件中心/调用组件-我的助手.png)

---

## 在工作流 / 对话型工作流中调用

进入工作流或对话型工作流编辑页，左侧节点面板顶部切换到**组件** Tab，即可看到所有可用组件列表，分为：

- **我的组件**：自定义创建的组件
- **业务流**：已发布的工作流
- **组件**：平台内置公共组件（google、bing、perplexity 等）

将组件从左侧列表拖拽到画布中，即可作为节点接入工作流。工作流与对话型工作流的调用方式完全一致。

![工作流-组件调用](/assets/组件中心/组件调用-业务流.png)

> **提示：** 若组件列表中显示"暂无数据"，请先前往**组件中心**完成对应组件的授权，或在**创建组件**中添加自定义组件。
