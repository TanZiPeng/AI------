# 角色权限管理

角色权限管理用于定义系统中不同角色所拥有的功能访问权限。管理员可通过顶部导航栏进入**用户中心 → 角色**页面，对角色进行查看、新增、编辑和权限配置。

---

![角色权限管理页面](/assets/组织架构/角色权限管理-1.png)

## 角色列表

角色列表展示当前系统中已创建的所有角色，每条记录包含以下信息：

| 字段 | 说明 |
|------|------|
| 角色名称 | 角色的显示名称 |
| 角色描述 | 对该角色职责或用途的简要说明 |
| 成员数量 | 当前被分配该角色的用户数量 |
| 创建时间 | 角色的创建日期 |
| 操作 | 支持对角色进行编辑或删除 |

---

## 新增角色

点击页面左侧的**新增角色**按钮，弹出新增角色弹窗。

![alt text](/assets/组织架构/角色权限管理-2.png)

填写以下信息后点击**确认**完成创建：

| 配置项 | 说明 |
|--------|------|
| 职务名称 | 填写角色的名称，需唯一 |
| 职务描述 | 选填，描述该角色的职责范围 |

---

## 搜索角色

点击图中筛选按钮即可输入手机号查询对应的角色用户
![搜索角色](/assets/组织架构/角色权限管理-4.png)

## 权限配置

点击勾选角色列表中某条记录左侧的勾选框，再点击**设置权限**按钮，右侧弹出权限配置面板。

![权限配置](/assets/组织架构/角色权限管理-3.png)

权限按功能模块树形展示，勾选即代表该角色拥有对应功能的访问或操作权限。配置完成后点击**保存**生效。

### 访问权限

配置面板顶部的**访问权限**字段，用于限定该角色可登录访问的系统入口，例如：

| 访问权限值 | 说明 |
|------------|------|
| AI控制台（Console） | 可登录并使用后台管理控制台 |
| 智能应用专项访问 | 仅可访问对外发布的智能应用，不可进入控制台 |

---

#### AIGC 应用

##### 应用中心

| 权限项 | 说明 |
|--------|------|

| 权限项 | 说明 |
|--------|------|
| 资料管理 | 管理AI-FAE 知识广场上传的文档等资料文件 |

| 权限项 | 说明 |
|--------|------|
| 私域知识库 | 访问私域知识库模块（父级权限） |

##### 组件中心

| 权限项 | 说明 |
|--------|------|
| 组件中心 | 访问用的 AI 组件（如工具函数等） |

---

#### 用户中心

| 权限项 | 说明 |
|--------|------|
| 公司 | 查看和编辑公司基础信息及登录页配置 |
| 部门 | 管理组织架构中的部门（新增、编辑、删除） |
| 角色 | 管理角色及其权限配置 |
| 用户 | 管理系统用户（新增、编辑、禁用、分配角色） |

---

配置完成后点击**保存**使权限生效。

---

## 删除角色

点击角色列表中某条记录右侧的**删除**按钮，系统将弹出确认提示。

> **注意：** 删除角色前请确保该角色下已无成员，否则将影响相关用户的系统访问权限。



---

## 为用户分配角色

角色创建完成后，可在**用户管理**页面中为具体用户分配对应角色，用户登录后将自动获得该角色所配置的权限。
