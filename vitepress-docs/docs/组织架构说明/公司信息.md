# 公司信息

公司信息页面用于配置平台的基础资料、品牌展示及登录页样式，填写完成后点击**保存**生效。

---
![公司信息配置页](/assets/组织架构/公司信息-1.png)
## 基础信息

| 配置项 | 说明 |
|--------|------|
| 深色 Logo | 用于深色背景下展示的品牌 Logo |
| 浅色 Logo | 用于浅色背景下展示的品牌 Logo |
| 国家/地区 | 公司所在国家或地区 |
| 公司名称 | 填写公司全称，支持选择**简称/正式名称**切换显示方式 |
| 地址 | 公司注册或办公地址，支持填写多个地址 |

---

## 主要联系人信息

| 配置项 | 说明 |
|--------|------|
| 名字 | 主要联系人姓名 |
| 邮箱 | 联系人邮箱地址 |
| 电话 | 联系人手机或固话号码 |

---

## 登录页信息

登录页信息用于定制用户登录界面的视觉风格与文案。

| 配置项 | 说明 |
|--------|------|
| 左侧宣传图 | 登录页左侧展示的宣传图片 |
| 浏览器 icon | 浏览器标签页显示的网站图标（favicon） |
| 浏览器标题 | 浏览器标签页显示的标题文字 |
| 左侧主标题 | 登录页左侧的主标题文案 |
| 左侧副标题 | 登录页左侧的副标题文案 |
| 右侧登录标题 | 登录表单区域上方的欢迎语或标题文案 |

---

## 保存

所有配置项填写完成后，点击页面底部的**保存**按钮使配置生效。
