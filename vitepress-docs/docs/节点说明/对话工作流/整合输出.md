# 整合输出

## 定义
是工作流（Workflow/ChatFlow）里最常用的变量合并/数据聚合节点，作用就是：把上游多个节点的输出、变量、文本、内容拼在一起，输出一个统一的结果。

## 适用场景：
- 把「用户提问」+「知识库回答」+「历史对话」拼在一起
- 把多个工具节点（搜索、查询、生成）的结果合并
- 把变量格式化，给下游大模型 / 回复节点使用
- 统一输出格式，避免下游节点收到杂乱数据

## 配置使用
**输入变量：** 添加输入变量（arg1、arg2、arg3...）

**代码：** 在代码区使用 {{ }} 引用变量并排版

**输出变量：** 输出string类型，节点自动将所有输入内容拼接成最终输出，使得最终结果仅有一个复制按钮。
