# 对话SQL节点

## 定义

对话SQL节点是对数据库进行新查询。，需要指定待操作的数据库表和输入内容，支持通过自然语言智能生成SQL语句。

---

## 配置使用

### 新增数据库

点击数据库右侧的 ```+``` 符号，再点击```新建数据库```按钮，输入符合要求的数据库名称与该数据库的描述，最后点击确认按钮进入数据库表结构设计，新增字段的标签页。

![对话SQL节点1](/assets/对话SQL/对话SQL-新增1.png)

**1. 新增数据库表结构：**存储字段名称、存储字段的描述、选择存储字段的数据类型，是否必要。

**2. 新增数据库导入：**新增表结构后，切换到数据导入页面，可以选择上传一份Excel或者CVS格式文档，选择表字段映射后，上传文档完成。

**3. 新增数据库预览：**导入数据表完成后，切换到表预览标签页，左上角是全局搜索框，下方是上传的数据列表，按分页展示，可支持全局展示

![数据库表预览](/assets/对话SQL/表预览.png)

### 添加数据库

新增的数据库弹窗，选择 ```添加``` 按钮，该数据库表会添加在节点的数据表下方，可支持删除添加

### 删除数据库

新增数据库弹窗，点击```删除``` 按钮，即可删除新增的数据库
