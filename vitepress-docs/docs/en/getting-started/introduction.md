# Introduction

The AI Knowledge Intelligence Platform is an AI platform focused on building intelligent agents (Agents). It is designed to help you flexibly define business processes visually and rapidly build intelligent agents using advanced AI technology to precisely address real business needs. The platform has a powerful built-in knowledge base that lets you easily build a dedicated intelligent Q&A assistant. In addition, the platform provides translation assistants, knowledge bases, workflows, conversational workflows, knowledge recycling, and other core features to help you fully control the building and optimization of AI applications.
