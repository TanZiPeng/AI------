# Knowledge Square

Users with the appropriate permissions will see the **Knowledge Square** menu in the top navigation bar.

## Library List

Knowledge Square contains multiple knowledge libraries, each corresponding to the knowledge recycling library of one Agent.

![Knowledge Square Entry](/assets/知识回收/知识回收-8.png)

Click to enter a library and view the sync history records.

![Knowledge Base Sync Records](/assets/知识回收/知识回收-9.png)

---

## Recall Testing

On the library detail page, you can run **Recall Testing** — enter different questions to see which knowledge segments are recalled. Use this to tune your embedding / rerank thresholds.

![Recall Testing](/assets/知识回收/知识回收-10.png)

---

## Record Management

After returning to the sync record list, the following operations are supported:

- **Delete sync record** — Delete an entire sync batch record
- **View Q&A details** — Click a sync record to view all Q&A message pairs from that recycling session
- **Delete individual Q&A** — On the detail page, delete a specific Q&A pair

![Q&A Detail Management](/assets/知识回收/知识回收-11.png)
